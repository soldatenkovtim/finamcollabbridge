# Finam Collab Bridge

Next.js 14 + React + TypeScript.

## Что нужно для работы в Cursor

### 1. Node.js

Нужна версия **Node.js 20** (LTS).

- Скачать: [nodejs.org](https://nodejs.org/)
- Или через nvm: `nvm install` (в папке проекта подхватится версия из `.nvmrc`)

### 2. Зависимости проекта

В корне проекта:

```bash
npm install
```

### 3. Запуск

```bash
npm run dev
```

Приложение откроется на **http://localhost:3000**.

### 4. Расширения в Cursor

При открытии проекта Cursor может предложить установить рекомендуемые расширения (ESLint, Tailwind CSS, Prettier) — их стоит поставить для удобства.

---

- `npm run build` — сборка
- `npm run start` — запуск собранного приложения
- `npm run lint` — проверка линтером
